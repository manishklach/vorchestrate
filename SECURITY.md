# Security Policy

vOrchestrate is an early prototype and research implementation. Security hardening is not yet complete.

## Reporting a Vulnerability

Please do not open public issues for suspected security problems.

Instead, report them privately to `manishklach@gmail.com` with:

- a clear description of the issue
- affected files or components
- reproduction details if available
- impact assessment if known

## Scope Notes

Because this repository is currently a prototype scaffold, some components are intentionally incomplete or illustrative. Even so, security issues in packaging, CI, scripts, or future integration surfaces should be reported responsibly.
